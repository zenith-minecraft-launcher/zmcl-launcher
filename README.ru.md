# **Zenith · Лаунчер Minecraft**

> Современный, полнофункциональный лаунчер Minecraft, построенный на Electron.

---

## ✨ **О проекте**

**Zenith** — это **все-в-одном лаунчер Minecraft**, объединяющий загрузку игры, управление версиями, поддержку модов, сетевую игру и ИИ-помощника. Он предлагает чистый и элегантный пользовательский интерфейс, поддерживает официальный / офлайн / Authlib методы входа, интегрирует популярные загрузчики модов, такие как Fabric, Forge, NeoForge и OptiFine, а также имеет встроенные функции P2P-сетевой игры на базе EasyTier и ИИ-помощника на базе DeepSeek (функции ИИ требуют спонсорской поддержки).

---

## 🚀 **Основные возможности**

### 🎮 **Запуск игры и управление версиями**
- Поддержка всех версий Minecraft: релизные, снапшоты, старые версии, версии ко Дню дурака
- Загрузка в один клик с автоматическим дополнением файлов ресурсов (Client / Assets / Libraries)
- Одновременное существование нескольких версий; переключение версий вступает в силу немедленно
- Умное обнаружение Java с автоматическими рекомендациями и автоматической загрузкой соответствующей версии Java

### 🔐 **Несколько методов аутентификации**
- ✅ **Официальный вход через Microsoft**: полный поток OAuth2, поддержка подлинных учётных записей Xbox
- ✅ **Оффлайн-вход**: пользовательское имя пользователя, запуск игры без интернета
- ✅ **Сторонний вход через Authlib**: поддержка пользовательских серверов аутентификации, подходит для офлайн-серверов

### 🧩 **Экосистема модов и пакетов ресурсов**
- Агрегированный поиск по платформам **Modrinth** и **CurseForge**
- Поддержка модов / пакетов ресурсов / шейдеров / датапаков / миров / модпаков
- Автоматическое разрешение зависимостей, загрузка и установка в один клик
- Улучшенные русские/китайские названия (получает локализованную информацию из MC百科)

### 🔧 **Установка загрузчиков модов в один клик**
- **Fabric** — лёгкий, высокосовместимый
- **Forge** — классический, проверенный временем загрузчик
- **NeoForge** — форк Forge, рекомендуется для новых версий
- **OptiFine** — оптимизация производительности и поддержка шейдеров
- Автоматическое обнаружение совместимости версий с предупреждениями о конфликтах

### 🌐 **Сетевая игра Тао-Ва (EasyTier)**
- **Не требуется публичный IP-адрес**, автоматическое переключение P2P-проброса / релея
- Механизм кодов комнаты: генерация кодов приглашений, друзья присоединяются в один клик
- Несколько узлов сообщества, низкая задержка, высокая стабильность
- Встроенная загрузка и управление ядром, готово к использованию сразу

### 🤖 **ИИ-помощник (DeepSeek)**
- Встроенный ИИ-помощник, отвечающий на вопросы, связанные с Minecraft
- Поддержка **потокового вывода** для ответа в реальном времени по одному символу
- Опциональный **режим глубокого размышления** и **веб-поиск**
- Поддержка **пользовательских моделей, совместимых с OpenAI**, без ограничений на использование
- Активируется через спонсорскую поддержку на 爱发电 (ifdian.net); поддерживает режим разработчика

### 🛠️ **Инструментарий**
- Открыть каталог игры / каталог версии / каталог журналов
- Резервное копирование и восстановление сохранений
- Очистка кеша, очистка старых журналов
- Диагностика сети, обнаружение среды Java

### 🎨 **Другие возможности**
- Переключение тёмной / светлой темы
- Пользовательские источники загрузки (официальный / BMCLAPI / собственное зеркало)
- Пользовательские аргументы JVM, выделение памяти
- Автообновление (на базе electron-updater)
- Визуализация прогресса загрузки с ясным управлением задачами
- Захват в реальном времени и экспорт журналов запуска

## 📦 **Быстрый старт**

### Требования к окружению
- **Node.js** ≥ 18
- Любой из **npm** / **pnpm** / **yarn**
- Windows 10+ / macOS 11+ / Linux (поддерживается AppImage)

### Локальная разработка

```bash
# 1. Клонировать проект
git clone <your-repo-url>
cd Zenith

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки
npm run dev
```

### Сборка релизных пакетов

```bash
# Сборка для текущей платформы
npm run build

# Или сборка для каждой платформы отдельно
npm run build:win      # Windows (установщик NSIS)
npm run build:mac      # macOS (.dmg)
npm run build:linux    # Linux (.AppImage)
```

Артефакты сборки будут выведены в каталог `dist-release/`.

## 🔒 **Безопасность и конфиденциальность**

- Учётные данные для входа пользователя (Microsoft Token, Authlib Token) хранятся только локально
- ИИ-чат по умолчанию использует API DeepSeek; данные хранятся только на устройстве пользователя
- Сетевая игра Тао-Ва использует протокол P2P; данные не хранятся на центральных серверах
- Данные пользователя не загружаются (за исключением проверок автообновления)

## 📝 **Руководство разработчика**

### Добавление нового инструмента в инструментарий

Отредактируйте `src/main/net/toolbox.js` и добавьте запись в массив `tools`:

```js
{
  key: 'my-tool',
  name: 'Мой инструмент',
  description: 'Описание инструмента',
  category: 'system',
  icon: '⚙️',
  async exec() {
    // Логика инструмента
    return { ok: true, message: 'Выполнение успешно' };
  }
}
```

### Добавление нового загрузчика модов

Расширьте метод `detectLoaders()` в `src/main/download/loader.js` и добавьте логику загрузки и распаковки в `installLoaderVersion()`.

### Пользовательские зеркала API

Измените массив `sources` в `src/main/download/sources.js`.
